## Init
```bash
git clone https://github.com/YenYuHsuan/hexo-theme-beantech.git ./hexo-beantech
cd hexo-beantech
npm install
```

## Modify
Modify `_config.yml` file with your own info.
Especially the section:
```yml
post_asset_folder: true
```
